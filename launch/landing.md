# resp-diff — Landing Page

## Headline

**See exactly what changed in your API. Right inside Chrome DevTools.**

## Subhead

One-click capture of API response bodies. Diff any two snapshots side-by-side with color-coded clarity. No external tools, no copy-paste, no leaving the browser.

## Benefit Bullets

1. **Zero-friction capture** — Click any request in the Network tab to snapshot its response body instantly. No manual copy, no switching windows.
2. **Color-coded JSON diff** — Side-by-side comparison highlights added, removed, and changed keys. Spot regressions in seconds, not minutes.
3. **Persistent & private** — Snapshots live in `chrome.storage.local` across browser sessions. Zero external dependencies, zero network calls, zero tracking.
